# Synthwave Courier: Last Cartridge

Аркадная HTML5 Canvas-игра в стиле synthwave / CRT / pixel arcade.

## Запуск

Открой `index.html` в браузере или запусти через Live Server в VS Code.

Структура проекта:

```text
index.html
styles/style.css
js/game.js
favicon.jpg
```

## Управление

| Действие | Клавиши |
|---|---|
| Газ / тормоз | W/S, стрелки, русская раскладка Ф/Ц/Ы/В |
| Поворот | A/D, стрелки |
| Нитро | Shift |
| Огонь / гараж у здания магазина | F / Enter |
| Выбор оружия | 1 / 2 / 3 |
| Мини-карта | M |
| Маршрут | Tab |
| Пауза | Space |
| Главное меню | Esc |
| Рестарт | R |
| Скрыть обучение | H |
| Музыка | N |

## Реализовано

- 5 уровней: обучение, доставка, сложная доставка, боевой уровень, босс.
- Контекстное обучение на первом уровне.
- Прогресс уровней и сохранение в `localStorage`.
- Монеты, магазин, покупка оружия и улучшений.
- Здание гаража на старте каждого уровня.
- Выбор активного оружия: пушка, дробовик, ракеты.
- Дроны, отдельные наземные роботы и финальный босс Petya.
- GPS-маршрут без диагональных линий – маршрут строится ортогонально по дорогам.
- ESC возвращает в главное меню.
- Исправлено поведение поворота после столкновений.
