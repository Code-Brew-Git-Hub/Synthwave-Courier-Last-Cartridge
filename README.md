# Synthwave Courier: Last Cartridge

HTML5 Canvas arcade game in synthwave / pixel-neon style.

## Запуск

Открой `index.html` через Live Server / локальный браузер из корня проекта.

Структура путей:

```text
index.html
styles/style.css
js/game.js
favicon.jpg
```

## Версия

`v0.17.1 · Ten Level Expansion`

## Управление

| Действие | Клавиши |
|---|---|
| Газ / тормоз | W/S, стрелки, русская раскладка |
| Поворот | A/D, стрелки, русская раскладка |
| Нитро | Shift |
| Огонь / вход в гараж | F / Enter |
| Выбор оружия | 1 / 2 / 3 |
| Мини-карта | M |
| Маршрут | Tab |
| Пауза | Space |
| Рестарт | R |
| Главное меню | Esc |

## Реализовано

- 10 уровней с прогрессом, выбором уровня и лучшими результатами.
- Первые 5 уровней сохранены как базовая кампания.
- Добавлены 5 новых крупных уровней: Skyway Spiral, Data Graveyard, Rainline Express, Glitch Citadel, Nexacom Core.
- Гараж на старте каждого уровня: покупка оружия и апгрейдов, выбор активного оружия, ремонт корпуса, заправка нитро.
- Активное оружие видно в HUD и в гараже.
- Стрельба доступна с уровня 2, сложность растёт постепенно.
- Роботы трёх типов визуально различаются: Scout, Tank, Turret.
- Robot-level использует настоящих роботов, а не только дронов.
- Boss fight Petya: аренный триггер, несколько фаз, финальный Overdrive.
- GPS-маршрут рисуется по дорогам и остаётся рекомендательным, а не обязательным.
- ESC открывает внутриигровое предупреждение, без браузерного confirm.
