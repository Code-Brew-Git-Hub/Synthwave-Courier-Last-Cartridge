# Synthwave Courier: Last Cartridge

> *1987 год. Корпорация NEXACOM планомерно гасит аркадные клубы по всему городу. Остался последний картридж — и ты единственный, кто может его доставить.*

Браузерная аркада на чистом Canvas без единой зависимости. Неоновые дороги, процедурно сгенерированный город, дроны на хвосте, хардкорный финальный босс и synthwave-саундтрек прямо в коде.

## Механики

**Движение и физика**
Машина управляется как аркадный автомобиль — скорость накапливается, есть инерция, повороты на высокой скорости широкие. Нитро расходует заряд буста, который постепенно восстанавливается. При столкновении со зданием скорость гасится, но направление поворота не инвертируется — это было баг, сейчас починено.

**Комбо**
Подбираешь бонусы подряд, не получая урон — множитель растёт до ×4. Теряешь HP или улетаешь в здание — комбо сбрасывается. Влияет на количество очков за всё.

**GPS-навигация**
Жёлтая стрелка наверху экрана ведёт не к финишной точке напрямую, а к следующему waypoint маршрута. Маршрут выстроен по дорогам, никаких диагоналей через всю карту.

**Glitch-зоны**
Розовые мерцающие пятна на дороге. Ехать можно, но каждую секунду снимают HP. Лучше объезжать, если есть время.

**Boost-пады**
Стрелки на дороге — наезжаешь, получаешь мгновенный выброс скорости и немного нитро. С комбо дают дополнительные очки.

**Угроза (Threat Level)**
Датчик в правом верхнем углу. Растёт, когда дроны вплотную — это индикатор опасности, не стелс-механика.

## Гараж(магазин)

Здание гараж стоит рядом со стартом на каждом уровне — зелёное свечение, подпись SHOP. Подъезжаешь и нажимаешь `Space`. Можно также открыть из главного меню через «Гараж» или через `Esc` прямо в игре.

| Предмет | Цена | Что даёт |
|---|---|---|
| 🔫 Передняя пушка | 30 | Прямой выстрел вперёд, кулдаун 0.4с |
| 💥 Дробовик | 55 | Веер из 5 снарядов, кулдаун 0.8с |
| 🚀 Ракетница | 80 | Самонаводящаяся ракета, кулдаун 1.8с |
| 🛡️ Броня +50 | 45 | Стартуешь со 150 HP вместо 100 |
| ⚙️ Тяжёлая броня | 75 | 200 HP + -25% к входящему урону |
| ⚡ Нитро ×2 | 40 | Двойной заряд, восстановление в 2 раза быстрее |
| 🔥 Нитро Turbo | 65 | Тройной заряд + скорость буста +20% |
| 📡 Радар дронов | 35 | Дроны на мини-карте с кольцом обнаружения, -15% к их скорости |

Деньги начисляются за прохождение уровней: `score / 80`, минимум 5 монет. Повторное прохождение одного уровня монет не даёт.

Купил несколько оружий — переключай клавишами `1` / `2` / `3`. В гараже есть кнопка EQUIP для каждого купленного ствола.

## Враги

**Дроны NEXACOM** — летающие ромбы, патрулируют дороги. При обнаружении игрока начинают преследование. Контактный урон. Можно сбить любым оружием.

**Наземные роботы** (уровень 4) — отдельная механика. Квадратные меховые машины, стреляют снарядами, держатся на дороге. У каждого свой HP-бар над головой. Уровень засчитан только когда уничтожены все.

**Bosс PETYA** (уровень 5) — появляется когда въезжаешь в арену, не по таймеру. 800 HP, две фазы:
- Фаза 1: медленнее, стреляет тройным залпом
- Фаза 2 (< 50% HP): ускоряется, дэш, пятерной залп, спавнит дронов  

После смерти открывает escape zone.

## Ранги и прогресс

По итогам уровня считается итоговый счёт:

```
Итог = бонус за время + бонус за HP + очки маршрута
```

| Ранг | Порог |
|---|---|
| C | < 4000 |
| B | 4000+ |
| A | 6500+ |
| S | 9000+ |

Лучший результат и ранг по каждому уровню сохраняются в `localStorage`. Видны на экране выбора уровня.

## Структура проекта

```
├── js/
│   └── game.js - механика игры
├── styles/
│   └── style.css -стилизация игры
├── favicon.jpg - иконка браузера
└── index.html - стартовый файл игры
```

Весь игровой движок в одном файле. Нет фреймворков, нет бандлера, нет TypeScript. Просто canvas 2D API.

Основные классы внутри `game.js`:

- `AudioEngine` — синтезирует synthwave прямо через Web Audio API. Баслайн на sawtooth, арпеджио на square, отдельные SFX для каждого события
- `RadioSystem` — контекстные реплики диспетчера и NEXACOM в левом нижнем углу
- `Particle` — частицы взрывов, подборов, выхлопа

## Технические детали

- **Рендер**: Canvas 2D с HiDPI-поддержкой (devicePixelRatio до ×2)
- **Физика**: собственная, покадровая, через `requestAnimationFrame`. dt-based, не framerate-locked
- **Карты**: процедурная генерация зданий через grid с проверкой пересечений с дорогами. Дроны, бонусы, glitch-зоны случайно размещаются по road-сегментам
- **Ввод**: двойная таблица маппинга — по `e.code` (layout-независимо) и по `e.key.toLowerCase()` (fallback с поддержкой кириллицы)
- **Сохранения**: `localStorage`, ключ `synthwaveCourierProgress`. Хранит уровень, монеты, купленное снаряжение, activeWeapon, bestByLevel
- **Аудио**: Web Audio API, инициализируется по первому взаимодействию. Генеративный синтез, никаких файлов

---

## Известные ограничения

- Аудио не запустится без взаимодействия с пользователем — это ограничение браузеров, не баг
- На очень маленьких экранах (< 360px) тач-контролы могут перекрываться
- Сохранения привязаны к домену/файловому пути — при переносе `localStorage` не перенесётся
